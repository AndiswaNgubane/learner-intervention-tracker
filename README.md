# learner-intervention-tracker
Learner Intervention Tracker

Author: Andiswa Ngubane
Date: July 2026

Overview

The Learner Intervention Tracker is a Python-based educational analytics tool designed to identify topic-specific learning gaps and support targeted academic intervention.

The system analyses learner performance across five Life Sciences topics and generates intervention reports for educators.

Features

* Reads learner data from a CSV file
* Calculates learner averages
* Assigns risk levels
* Identifies topics requiring intervention
* Identifies the most challenging topic across the class
* Identifies the top-performing learner
* Generates class-level intervention statistics

Technologies Used

* Python
* CSV Module
* Functions
* Dictionaries
* Lists
* Loops
* Conditional Statements

Topics Assessed

* DNA: The Code of Life
* Meiosis
* Reproduction in Vertebrates
* Human Reproduction
* Genetics

This project assists educators in identifying learner performance gaps and planning targeted intervention strategies to improve academic outcomes.
